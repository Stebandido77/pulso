# Examples

This folder will contain Jupyter notebooks demonstrating realistic uses of `pulso`. Populated in Phase 6.

Planned notebooks:

- `unemployment_series.ipynb` — Build the national unemployment rate series 2018-present, compare against DANE's official monthly figures.
- `informal_employment.ipynb` — Track informality, with the caveat about the 2021 methodological change.
- `replicate_official.ipynb` — Step-by-step reproduction of one DANE press-release figure, walking through expansion factor application.
